Chapter 9.2, Problem 9E
Problem

Suppose there are three roads from city A to city B and five roads from city B to city C.

a. How many ways is it possible to travel from city A to city C via city B?

b. How many different round-trip routes are there from city A to B to C to B and back to A?

c. How many different routes are there from city A to B to C to B and back to A in which no road is traversed twice?

Step-by-step solution
  1. Step 1 of 4

    It is given that there are 3 roads from city A to city B, and 5 roads from city B to city C.

  2. Step 2 of 4

    (a) If one has to travel from city A to city C via city B, the number of ways in which one can travel from city A to city B is C (3, 1), and the number of ways in which one can travel from city B to city C is C (5, 1).

    Thus, the total number of ways in which one can travel from city A to city C is C (3, 1)C (5, 1), by the rule of multiplicity.

    The required answer is

    = 15

  3. Step 3 of 4

    (B) We have to find the number of different round trip routes from A – B – C – B – A

    The number of ways to travel from A to B is C (3,1)

    The number of ways to travel from B to C is C (5,1)

    The number of ways to travel from C to B is C (5,1)

    The number of ways to travel from B to A is C (3,1)

    The total number of ways of the round trip route A – B – C – B – A is

    = 225

  4. Step 4 of 4

    (c) We have to find the number of different routes from city A to B to C to B and back to A

    The number of ways to travel from A to B is C (3, 1)

    The number of ways to travel from B to C is C (5, 1)

    The number of ways to travel from C to B is C (4, 1)

    The number of ways to travel from B to A is C (2, 1)

    Since no road is traveled twice, we understand that if one route from B to C is selected in C (5,1) ways, then to travel from C to B the route already selected is not to be considered, so the number of ways from C to B is from the remaining 4 routes.

    The number of ways to travel from C to B is C (4, 1)

    Similarly, the number of ways to travel from B to A is C (2, 1)

    The total number of different routes is

    = 120
