Chapter 9.2, Problem 4E
Problem

In, use the fact that in baseball’s World Series, the first team to win four games wins the series.

How many ways can a World Series be played if no team wins two games in a row?

Step-by-step solution
  1. Step 1 of 2

    Consider that there is baseball’s World Series.

    If one team to win 4 games wins the series.

    The objective is to find the number of ways can a World Series be played if no team wins two games in a row.

  2. Step 2 of 2

    If no team wins two games in a row that the teams win in alternative games.

    If A wins the first game, then the second game is won by B, and the third game is won by A, the fourth game by B and so on.

    Similarly, if B wins the first game, the second game is won by A and so on.

    Thus, there are 2 ways in which the teams win alternately.
