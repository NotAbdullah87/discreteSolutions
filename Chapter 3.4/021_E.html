Problem

Indicate whether the arguments are valid or invalid. Support your answers by drawing diagrams.

Exercise

All people are mice.

All mice are mortal.

Step-by-step solution
  1. Step 1 of 3

    All people are mice

    All mice are mortal

    All people are mortal

  2. Step 2 of 3

    This argument is valid

  3. Step 3 of 3

    The major premise says the set of people is included in the set of the mice. The major premise says that the set of mice is included in the set of mortals.

    Assuming that both of the premises are true, it must follow that the set of people is included in the set of mortals

    Since it is impossible for the conclusion to be false if the premises are true, we know that the argument is true

    Thus, this is a valid argument
