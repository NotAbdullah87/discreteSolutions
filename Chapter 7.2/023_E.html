    (a)

    In a one-to-one function, each element of the domain maps with one element of the co-domain.

    No two or more elements of the domain map with the same element of the co-domain.

    Let the elements of the domain to be analyzed are two sets:

    But

    There can be infinitely many inputs with the same output as long as they have the same number of elements in the set.

    Thus, two or more different values of input plugged into the function

    may give the same value.

    Since, same output is possible for different values of input;

    Therefore, it is proved that function is NOT one-to-one function.

      (b)

      In a onto function, all elements in the co-domain set have corresponding values in domain set such that when the value in domain set is plugged into the function formula, the co-domain element is obtained.

      So, if is onto function, then it must be possible to obtain every integer that is possible by plugging in various kinds of sets as input into the function.

      because number of elements can’t be negative.

      Therefore, it is impossible to obtain negative integers using this function.This means that all elements in the co-domain set can’t be obtained by using the function.

      Therefore, the function is NOT onto function.
