Chapter 2.5, Problem 29E
Problem

Find the decimal representations for the integers with the 8-bit representations given in 27–30.

11110010

Step-by-step solution
  1. Step 1 of 2

    The given number is 11110010.

    Since, the first digit of the number is 1, it is 8 bit representation of a negative number.

    To find the number, we replace all 0 with 1 and vice-versa.

    Thus, the binary number now becomes 00001101.

    Comment
    • Step 2 of 2

      Now, we add 1 to the number to get the original number.

      Thus, the binary number is 00001110.

      To convert the binary number to decimal notation, we start from right and add the product of each digit with 2digit’s place –1.

      Hence, the number in decimal form is –14.
