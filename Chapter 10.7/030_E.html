Chapter 10.7, Problem 30E
Problem

Prove that if a connected, weighted graph G is input to Algorithm 10.7.4 (shown below), the output is a minimum spanning tree for G.

Step-by-step solution
  1. Step 1 of 1

    The input of the algorithm is a connected graph G

    According to the algorithm remove an edge from G which is having a maximal weight and at each stage G must be connected.

    By repeating this process for each m, we get a spanning tree having minimum weight.

    Hence the output is a minimal spanning tree.
