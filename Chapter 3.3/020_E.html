Chapter 3.3, Problem 20E
Problem

Recall that reversing the order of the quantifiers in a statement with two different quantifiers may change the truth value of the statement—but it does not necessarily do so. All the statements in the pairs below refer to the Tarski world of Figure. In each pair, the order of the quantifiers is reversed but everything else is the same. For each pair, determine whether the statements have the same or opposite truth values. Justify your answers.

a. (1) For all squares y there is a triangle x such that x and y have different color.

(2) There is a triangle x such that for all squares y, x and y have different colors.

b. (1) For all circles y there is a square x such that x and y have the same color.

(2) There is a square x such that for all circles y, x and y have the same color.

  2. Step 2 of 6

    a.

    (1)

    The objective is to determine whether the statements have the same or opposite truth values.

    For all squares there is a triangle such that have different color.

    From the figure, observe that squares are light grey and choose a triangle then the square and triangle have different color.

    Square has dark grey and choose triangles then the square and triangle have different color.

    Square has blue color and choose triangles then the square and triangle have different color.

    Thus, for every square there is at least one triangle that has a different color.

    Therefore, the truth value of the statement is true.

  3. Step 3 of 6

    (2)

    There is a triangle such that for all squares have different color.

    From the figure, observe that for triangle there is no square that have different color.

    For triangle there is no square that have different color.

    For triangle there is no square that have different color.

    Thus, there is no triangle that has a different color from all squares.

    Therefore, the truth value of the statement is false.

  4. Step 4 of 6

    From the above two results, observe that by switching the universal and existential quantifiers in a statement, the truth value does not always remain the same.

    Therefore, the truth value of the statement is true.
