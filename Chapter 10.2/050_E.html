Step-by-step solution
  1. Step 1 of 2

    Suppose G is a bipartite graph with the partition of vertices given by and having a circuit.

    Suppose G has a circuit with an odd number of edges.

    Since G is bipartite, every path of minimum length from to is 2.

    Thus, to complete a circuit, it is required to traverse a path of length 2n going from the vertex set to itself or from the vertex set to itself.

    It follows that a path of odd number of edges or odd degree will go from one vertex to the other vertex set in a bipartite graph.

    This confirms that a circuit cannot be completed with odd number of edges in a bipartite graph.

  2. Step 2 of 2

    Conversely, suppose there is a path with odd number of edges in a bipartite graph.

    For a path from one vertex set to the same vertex set of minimum length, a length of at least 2 edges needs to be traversed. The path of length 2k + 1 traverses from one vertex set to the other vertex set.

    As there is no edge between any pair of vertices in one vertex set, it confirms that the path of length 2k + 1 cannot be a circuit.

    Thus, by the contra positive statement, it follows that a circuit must not be of length 2k + 1 in a bipartite graph.
