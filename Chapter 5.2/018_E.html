    Mathematical Induction: This is a method of proving theorems in which the theorem is shown to be true for one case and then it is assumed that the theory holds true for an arbitrary case. Next by using the assumed case, the objective is to show that the theorem holds true for the immediate next case that comes up in the series. Once, this is proved, by virtue of having proved already that it is true for one case, the conclusion can be made that the theorem holds for all of the series.
