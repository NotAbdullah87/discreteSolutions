Chapter 9.4, Problem 39E
Problem

What is the largest number of elements that a set of integers from 1 through 100 can have so that no one element in the set is divisible by another? (Hint: Imagine writing all the numbers from 1 through 100 in the form 2k • m, where k ≥ 0 and m is odd.)

Step-by-step solution
  1. Step 1 of 6

    Let us consider the integers from 1 to 2n, for n = 1, 2, 3

    For

    Among 1 to 2 integers, the set of largest number of elements such that no one element in the set is divisible by another is 1.

  3. Step 3 of 6

    Consider for

    Among 1 to 4 integers, the set of largest number of elements such that no one element in the set is divisible by another is 2.

  4. Step 4 of 6

    Consider for

    Among 1 to 6 integers, the set of largest number of elements such that no one element in the set is divisible by another is 3.

    Therefore, it is observed that in a set of integers from 1 to 2n, set of numbers in the top half of the range satisfies the required property.

  5. Step 5 of 6
    Any number in the range 1 to n, if included in the set, its double is already present in the set.
  6. Step 6 of 6

    Or in the integers 1 to 2n, the set have no element which is divisible by another.

    Therefore, a set of integers from 1 through 100 can have elements which have no element which is divisible by another.

    Therefore, the largest number of elements is 50.
