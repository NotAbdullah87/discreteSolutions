    The merge short algorithm divides the given array into possible equal subarrays, and then all the subarrays are sorted in the ascending order, that is, in the alphabetical order by using the merge algorithm. In the given question, the array is divided into subarrays at each stage till the subarray has only one element in it. An array having one element is already sorted, so the subarrays are then merged using merge algorithm, which merges the element at each stage in ascending order. Thus, the final array is obtained which is in alphabetical order.

    • Step 2 of 5

      While merging the arrays, at each stage, place each subarray in two rows, one at the left and one at the right. Along the bottom, set required number of positions into which the elements of the subarray will be moved. Now, one-by-one, bring down the elements from the left most position of the two rows. At each stage compare the elements currently at the left most position and move the element which is smaller down into the next position in the final array as a whole. It is to be noted that, when the two elements are same, use the convention, say, the element in the left array moves to the next position in the final array.
